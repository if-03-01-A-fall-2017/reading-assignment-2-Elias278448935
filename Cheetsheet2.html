Attributes

All HTML elements can have attributes

Attributes provide additional information about an element

They're always specified in the start tag

They usually come in name/value pairs: name="value"

CSS rule

selector - declaration

h1 {property:value; property:value;}

descendant selector (space), The descendant selector matches all elements that are descendants of a specified element.

child selector (>), he child selector selects all elements that are the immediate children of a specified element

adjacent sibling selector (+), The adjacent sibling selector selects all elements that are the adjacent siblings of a specified element.

general sibling selector (~), The general sibling selector selects all elements that are siblings of a specified element.

Fonts

Serif: fonts have small lines at the ends on some characters

sans-serif: sans=without, no lines

font-family: should hold some font names as an option if the browser doesn't support the 1st font

Example:

p {

font-family: "Times New Roman", Times, serif;

}

Font Size

-absolute size:

  • sets the text to a specified size
  • user can't change the text size
  • useful when the size of the output is known

font-size: small;

-relative size:

  • size set relative to surrounding elements
  • user can change the text size

font-size: smaller;
